Página Lista de Tarefas (To-Do List)

Este projeto é uma Lista de Tarefas (To-Do List) totalmente interativa, desenvolvida com HTML, CSS e JavaScript puro, focada em oferecer uma experiência fluida, moderna e dinâmica ao usuário. A interface conta com animações, ícones funcionais e um sistema completo de persistência de dados usando LocalStorage.
